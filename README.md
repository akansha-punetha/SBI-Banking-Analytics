# SBI Banking Analytics Dashboard

## Overview

This project demonstrates an end-to-end Banking Data Analytics solution using Python, SQL, and Power BI. It focuses on analyzing customer, account, transaction, and loan data to generate business insights through interactive dashboards.

## Technologies Used

- Python (Pandas, NumPy)
- SQL (MySQL)
- Power BI
- Jupyter Notebook
- CSV Datasets
